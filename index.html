Fluid Flow Simulator - 3D

A 3D Flow simulator written in Python, using the Lattice Boltzmann Method
instead of the traditional Navier-Stokes equations to simulate fluid flow.

The animation on the left is a simulation of a free jet flowing into a box-shaped region.
A snapshot of the velocity field is captured every nth time-steps to produce this animation.
The snapshots are taken along a planar slice of the 3D Domain.
Turbulence is modeled by using the Smagorinsky sub-grid scale model.

The user can apply Dirichlet or Neumann boundary conditions.
Given a boolean mask of the domain, the flow simulator can handle any type of geometry.

Fluid Flow Simulator - 2D

A 2D Flow simulator written in MATLAB, using the Lattice Boltzmann Method
instead of the traditional Navier-Stokes equations to simulate fluid flow.

The predecessor to the 3D LBM code.
This implementation can take a list of coordinates or an .stl file to
generate the simulation domain.

Note that a 2D simulation is less computationally expensive, which means
a larger domain can be simulated.

AI Telegram Chatbot

An AI chatbot powered by OpenAI's GPT-3, written in Python.
The user can select different AI engines to generate better responses.
Each query and response uses up tokens.
AI token usage is calculated and tracked, and displayed at the request
of the user.

This AI chatbot is useful in social events, should the user finds
themselves with nobody to talk to. This AI chatbot will happily chat,
no questions asked.

Job Board Scraper

A Job Board Scraper written in Python.
The user specifies Job Title and Location in the Terminal.
The script will send multiple requests to the job board and parse
the responses using Beautiful Soup.

The scraped job postings will then be exported to excel
using the Pandas library.
